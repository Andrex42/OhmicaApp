# OhmicaApp

Ohmica App è una HMI remoto per impianti Siemens Based. L'app Ohmica è scritta in Flutter per tutti i dispositivi sia Windows, Mac, Android e IOS. Ho lavorato per questo progetto dal 2024 al 2025. Backend fatto da API sviluppate in Python e db MySql. Il mio ruolo nel progetto era principalmente l'occuparmi del backend development.
